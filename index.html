List-style-position xassəsi listlərin onlar tutan xəyalı divin içərisində və ya xaricində görünməsini göstərir. O, iki dəyərdən birinə malik ola bilər:

List-style-position: outside

  • CSS (Cascading Style Sheets) HTML və XHTML işarələmə dillərində yazılmış veb-səhifələrin təsviri və tərtibatı üçün istifadə edilir. CSS sənədin xarici görünüşünün, işarələmə dilinin istifadəsi ilə yazılmış, formal təsvir dili kimi tanınır.
  • Adətən HTML və XHTML işarələmə dillərində yazılmış veb-səhifələrin təsviri və tərtibatı üçün istifadə edilir, lakin istənilən digər XML sənədlərə, məsələn, SVG və ya XUL-a da tətbiq edilə bilər.
  • CSS standartından heç bir məhdudiyyət qoyulmadan istənilən serverdə istifadə etmək olar. CSS kodları, HTML kodlarının içinə yazıla bildiyi kimi xarici bir CSS faylı yaradılaraq da əməliyyat yaradıla bilir.
  • Bu işləri standart üsulla da yerinə yetirmək olar, yəni HTML dilində hər bir abzası ayrıca əmrlər yığımı ilə təsvir etmək mümkündür. Bu halda proqramın yazılması prosesi həddən artıq çətinləşəcək, sənədin ölçüsü isə çox böyük olacaq. Bu işi sadələşdirmək üçün səhifələrə CSS standartında yerinə yetirilmiş xarici fayl qoşulur və həmin faylda istifadə olunmuş xüsusi makrodil vasitəsilə səhifə birdəfəlik formatlaşdırılır. Başqa sözlə, CSS faylı burada şablon kimi istifadə edilir ki, onun köməyilə HTML sənədlərində mətnlər, cədvəllər və digər elementlər formatlaşdırılır. CSS standartından heç bir məhdudiyyət qoyulmadan istənilən serverdə istifadə etmək olar.
